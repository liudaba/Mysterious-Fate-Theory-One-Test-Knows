# ✨ 玄机命理 - 洞悉天机 ✨

传统命理学软件，提供专业的命理分析服务。提供 **Python GUI版** 和 **网页版** 两个版本，功能完全一致，相同输入产生相同输出。

---

## 📋 功能介绍

### 🔮 算命大师
- 八字命理分析：根据生辰八字计算命局
- 五行解读：分析五行旺衰、喜用神
- 流年运势：当年运势预测

### 📅 黄道吉日
- 择日择时：查询适合各类事项的吉日
- 事项吉日查询：婚嫁、搬家、开业等吉日推荐

### 📖 老黄历
- 每日宜忌：今日宜做与忌做事项
- 农历信息：干支、农历日期、冲煞信息
- 吉神凶煞：每日吉神与凶煞方位

### 💑 婚姻配对
- 生肖配对：基于传统命理的生肖契合度分析
- 契合度评分：三合、六合加分；六冲、六害减分

### 🌸 桃花运
- 桃花星分析：基于《三命通会》桃花星理论
- 一生桃花运势：18-58岁桃花运时间段预测
- 桃花强度与质量：正缘桃花、良缘桃花等分类

## 💻 安装说明

### Python GUI 版本

**环境要求：**
- Python 3.6 或更高版本
- Tkinter（Python自带，无需额外安装）

**验证Python安装：**
```bash
python --version
```

### 网页版本

**环境要求：**
- 任意现代浏览器（Chrome、Edge、Firefox、Safari等）
- 无需安装任何依赖

## 🚀 使用教程

### Python GUI 版本

**方式一：双击启动（推荐）**
1. 双击 `玄机命理.vbs` 文件
2. 程序将自动启动，无控制台窗口

**方式二：命令行启动**
```bash
# 带控制台窗口
python a1.py

# 无控制台窗口
pythonw a1.py
```

**使用步骤：**
1. 程序启动后进入主界面
2. 点击顶部导航栏选择功能模块
3. 填写所需信息（生日、性别等）
4. 点击分析按钮获取结果

### 网页版本

**启动方式：**
1. 双击 `玄机命理.html` 文件
2. 浏览器自动打开应用

**使用步骤：**
1. 在首页选择功能卡片
2. 填写所需信息
3. 点击分析按钮查看结果
4. 点击"返回首页"可切换其他功能
